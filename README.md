# 🧾 Tamil Form Guidance Web Application

## Overview
This project is a **Tamil-based web application** designed to help users understand and correctly fill government, banking, and postal forms. It provides **step-by-step guidance using text and audio support**, making it especially useful for non-English speakers and rural users.

---

## Objective
The main objective of this project is to:
- Reduce errors in form filling  
- Provide clear guidance in **Tamil language**  
- Assist users with **low technical knowledge**  

---

## 🏗️ System Modules

### 🏤 Postal Services
- PLI (Postal Life Insurance)
- PPF (Public Provident Fund)
- Savings schemes
- Deposit & Withdrawal forms
- Loan-related forms

### 🏦 Banking Services
- Account opening forms
- Deposit and withdrawal forms
- PPF-related forms
- Supports banks like SBI, Canara Bank, Indian Bank

### 🏛️ Government Services
- Income Certificate
- Driving License
- First Graduate Certificate

---

## Features
- 📖 Tamil text-based explanation for each field  
- 🔊 Dual audio support:
  - Audio 1: Purpose of the form  
  - Audio 2: Step-by-step filling instructions  
- 📱 Responsive design (works on all devices)  
- 🔄 Navigation with Previous / Next buttons  

---

## Tech Stack
- HTML  
- CSS  
- JavaScript  

---

## Working Process
1. User opens the application  
2. Selects a service category (Postal / Banking / Government)  
3. Chooses a form  
4. Reads Tamil instructions or listens to audio guidance  
5. Fills the form correctly based on guidance
